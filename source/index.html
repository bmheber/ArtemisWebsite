PORTFOLIO

Association Rule Mining

Association rule mining in conjunction with the apriori algorithm is a common technique used by large retailers to analyze customer purchasing behavior. The rules illustrate what items are frequently bought together. In this particular example, we used data published from Instacart- the online grocery store.

This project was built using R and Shiny. Check out it out here https://lbrandgroupgo.shinyapps.io/Lbrands/

Topic Modeling

The Latent Dirichlet Allocation (LDA) model is a statistical model that sorts documents into topics. This technique is incredibly useful as LDA allows the user to understand a large amount of data without actually reading every document.

For this project we used the Enron company's emails to attempt to find a topic of fraud. The next steps of this project will be to use SpaCys tagging of speech capability to flag the nouns and perform entity extraction to find names and addresses.
